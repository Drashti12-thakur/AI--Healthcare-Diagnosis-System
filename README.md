# 🧠 AI Healthcare Diagnosis & Medicine Finder System

## 📖 Overview
This project aims to reduce medical misdiagnosis and improve medicine accessibility using Artificial Intelligence.

It consists of two main modules:
1. Disease Classification System
2. Medicine Availability & Alternative Finder

---

## 🧠 Module 1: Disease Classification System

### 💡 Description
An AI-based system that analyzes patient symptoms, test results, and medical history to predict possible diseases.

### ⚙️ Technologies Used
- Supervised Learning
- Multi-class Classification
- Python (Flask/Django)
- Machine Learning Algorithms
- XAI (Explainable AI)

### 🎯 Features
- Predicts top probable diseases
- Provides confidence scores
- Explains predictions (XAI)
- Maintains logs for legal/audit use

### 🌍 Impact
- Reduces misdiagnosis
- Supports doctors in decision-making
- Improves clinical accountability

---

## 💊 Module 2: Medicine Availability & Alternative Finder

### 💡 Description
Helps users find nearby pharmacies and suggests alternative medicines if unavailable.

### ⚙️ Technologies Used
- APIs & Database
- Location Services
- Basic ML Recommendation

### 🎯 Features
- Nearby medicine search
- Alternative suggestions
- Emergency support

### 🌍 Impact
- Saves time in emergencies
- Improves healthcare accessibility
